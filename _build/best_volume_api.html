Best Volume API¶

The hotspots.extraction module contains classes to extract valuable information from the calculated Fragment Hotspot Maps.

The main fields can be broken due using a fixed volume or fixed score mode.

This approach enables a ranking of ligand sized cavity descriptions to be extracted. For example, these grids can be used to generate pharmacophores.

The main classes of the hotspots.best_volume module are:
-HotspotFeatures -ExtractHotspot -HotspotCreator
