Hotspot Pharmacophore API¶

The hotspots.pharmacophore module contains classes for the conversion of Grid objects to pharmacophore models.

The main classes of the hotspots.pharmacophore module are:

  • hotspots.pharmacophore.PharmacophoreFeature
  • hotspots.pharmacophore.PharmacophoreModel
