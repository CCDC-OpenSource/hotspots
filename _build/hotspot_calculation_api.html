get_selectivity_map(other)[source]¶

Generate maps to highlight selectivity for a target over an off target cavity. Proteins should be aligned by the binding site of interest prior to calculation.

High scoring regions of a map represent areas of favourable interaction in the target binding site, not present in off target binding site

Parameters:other – a hotspots.hotspot_calculation.HotspotResults instance
Returns:a hotspots.hotspot_calculation.HotspotResults instance
